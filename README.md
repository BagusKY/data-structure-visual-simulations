# Visualisasi Struktur Data (Python + Tkinter)

Project ini merupakan visualisasi interaktif dari beberapa konsep struktur data menggunakan Python dan Tkinter.

## 🎯 Tujuan
Membantu memahami cara kerja struktur data melalui simulasi visual dan animasi.

---

## 📦 Fitur / Case

### 1. Printer Queue (FIFO)
Simulasi antrian dokumen printer menggunakan Queue.

### 2. Hot Potato (Queue Simulation)
Simulasi permainan rotasi menggunakan Queue.

### 3. Hospital (Priority Queue)
Pasien dengan prioritas berbeda diproses berdasarkan urgensi.

### 4. BFS Graph Traversal
Visualisasi traversal graph menggunakan Breadth-First Search.

### 5. Airport Runway Scheduling
Simulasi penjadwalan pesawat dengan kombinasi Queue & Priority.

---

## 🏗️ Struktur Project

- `core/` → logic struktur data
- `components/` → elemen visual (box, node)
- `animations/` → sistem animasi
- `cases/` → implementasi tiap kasus
- `main.py` → entry point
